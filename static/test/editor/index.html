Quick start

  1. Have a Frothy-flashed ESP32 plugged in over USB. If you don't, hit the flasher first.
  2. Hit Connect. Pick your board from the WebSerial picker.
  3. Edit the sketch on the left. Hit Send buffer (or ⌘+Enter) to run the whole sketch on the chip. Hit Send line (or ⇧+Enter) to run just the current line.
  4. Save stores the sketch in your browser. Download .fr exports it as a file you can later send with frothy send.

Web Serial requires a recent Chrome or Edge on desktop. Firefox and Safari are not supported.

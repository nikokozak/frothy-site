---
title: "OldFroth"
description: "Archive policy for the original Frothy language and site."
---

OldFroth is the original Frothy language and documentation set. It remains useful
as history and reference material, but it is no longer the current product
described by this site.

The current language is Frothy.

The archive cutover is still in progress. Until the old site is moved into its
final archive location, this page is the public marker that old Frothy material
must be labeled OldFroth before it is linked from the new site.

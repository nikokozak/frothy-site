## Quick start

1. Plug in a [Frothy-flashed ESP32](/flash/) over USB.
2. Choose **Connect** and select the board from the browser's serial picker.
3. Use **Run File** to run the whole sketch, or place the cursor in one complete form and use **Run Form**.
4. Edits autosave locally in this browser. Use **Download .fr** when you want a file copy.

Web Serial requires a desktop Chrome, Edge, or Opera browser.
